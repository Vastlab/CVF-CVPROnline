This Plugin display date and time of all timezones (Server time adjusted). Easy to use with shortcodes.

== Description ==

This Plugin allows you to display time and date from all timezones. The setup page generates the shortcode you like. Just copy & paste this code on every place you want. The time is adjusted to the server time.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings -> World Clock screen to generate the shortcodes.
1. Copy the shortcode
1. Paste the shortcode where ever you want.
